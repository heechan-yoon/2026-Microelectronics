## 2. 🧱 Experiment 1: 다이오드 V-I 특성과 오차 분석

### 📊 측정 결과 (Theoretical vs Simulated/Measured)
*표 내부 수식 렌더링 오류를 방지하기 위해 띄어쓰기를 적용했습니다.*

| Input Voltage (V) | Theoretical $I_D$ (mA) | Simulated $I_D$ (mA) | Error (%) |
| :--- | :--- | :--- | :--- |
| -5.0 ~ 0.0 | 0.00 | 0.00 | 0% |
| 1.0 | 3.00 | 0.15 | 95.0% |
| 2.0 | 13.00 | 11.69 | 10.08% |
| 3.5 | 28.00 | 25.08 | 10.43% |
| 5.0 | 43.00 | 38.46 | 10.56% |

> **Note:** $V_{in}$이 문턱 전압 이하일 때는 다이오드가 OFF 상태이므로 전류가 흐르지 않음을 확인했습니다.

### 🔍 트러블슈팅 및 고찰 (Error Analysis) ⭐
1. **모델의 한계:** 상수 전압 모델은 전류 변화에 따른 전압 강하를 무시하므로 고전압 영역에서 오차가 발생함.
2. **벌크 저항:** 실제 다이오드의 내부 저항 성분이 전류 흐름을 방해하여 이론값보다 낮은 전류가 측정됨.

## 3. 📉 Experiment 2: 다이오드-RC 회로의 파형 성형 특성

### 📐 회로 설계 및 이론적 배경
* **시정수:** $\tau = R \times C$
* **동작:** 다이오드의 정류 작용과 커패시터의 충/방전 시간을 이용한 신호 처리.

### 📊 주파수 및 커패시턴스 변화에 따른 출력($V_{out}$) 분석

| 주파수 ($f$) | 커패시턴스 ($C$) | 동작 분석 (Region) | 결과 요약 |
| :--- | :--- | :--- | :--- |
| **10 Hz** | 0.1 / 1.0 $\mu F$ | Discharge Dominated | 주기($T$)가 길어 방전이 충분히 일어남 |
| **10 kHz** | 0.1 $\mu F$ | Fast Vibration | 방전 시간이 부족하여 높은 전압 유지 |
| **10 kHz** | **1.0 $\mu F$** | Peak Detecting | 시정수($\tau$) 증가로 매우 안정적인 DC 출력 |

### 🔍 트러블슈팅 및 고찰 (Error Analysis) ⭐
1. **주파수 영향:** 주파수가 높아지면 주기($T = 1/f$)가 짧아져 시정수 대비 방전 시간이 부족해지므로 리플이 감소함.
2. **커패시터 영향:** $C$ 값이 커질수록 에너지 저장 용량이 커져(시정수 증가) 출력 전압이 더 일정하게 유지됨(Clamping 효과).
