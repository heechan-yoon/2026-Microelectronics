# ⚡ [Week 5] 전파정류기(Full-Wave Rectifier) 분석

> **과목명:** 2026-1 전자회로1 실습  
> **실험 일자:** 2026.03.31 (Week 5)  
> **사용 장비:** Breadboard, Oscilloscope (TDS 2012C), Function Generator (AFG 1022), SPICE  

---

## 1. 🎯 프로젝트 목표
* **정류 동작 확인:** 브리지 전파정류기 회로에서 입력 신호의 음의 반주기가 양의 방향으로 뒤집혀 출력되는지 확인.
* **평활 효과 분석:** 부하 커패시터($C_L$)를 연결했을 때 출력 전압의 리플(ripple)이 어떻게 변화하는지 관찰.
* **주파수 영향 분석:** 입력 주파수 변화에 따라 출력 파형의 안정성과 평균 출력 전압이 어떻게 달라지는지 비교.
* **비교 분석:** 실제 측정값과 SPICE 시뮬레이션 결과 사이의 차이와 그 원인을 분석.

---

## 2. 🧱 브리지 전파정류기 기본 동작 분석

### 📊 측정 조건 (Setup)
* **회로 구성:** Full-Wave Rectifier (Bridge Rectifier)
* **부하 저항:** $R_L = 1\,k\Omega$
* **커패시터 조건:** No capacitor / $1\\mu F$ / $10\\mu F$
* **입력 주파수:** 10 Hz, 1 kHz, 10 kHz

| 주파수 ($f$) | 커패시터 조건 | 출력 파형 특성 | 결과 요약 |
| :--- | :--- | :--- | :--- |
| **10 Hz** | No cap | Pulsating DC | 정류는 되지만 리플이 크게 나타남 |
| **1 kHz** | No cap | Pulsating DC | 10 Hz보다 더 촘촘한 맥동 출력 확인 |
| **10 kHz** | No cap | Pulsating DC | 정류 출력의 주기가 더 짧아짐 |

> **Note:** 커패시터가 없는 경우 출력은 정류된 맥동형 전압으로 나타나며, 입력의 음의 반주기가 양의 방향으로 뒤집혀 전파정류 특성이 확인된다.

![KakaoTalk_20260331_115319248_02](https://github.com/user-attachments/assets/64e94b23-0bd5-4df4-8db5-bad519f0335a)

### 🔍 트러블슈팅 및 고찰 (Error Analysis) ⭐
1. **전압 강하 존재:** 브리지 정류기에서는 전류 경로마다 두 개의 다이오드가 도통하므로 실제 출력 전압은 이상적인 계산보다 더 낮게 나타날 수 있다.
2. **맥동 출력 특성:** 커패시터가 없을 때는 출력이 DC가 아니라 정류된 맥동 파형이므로 리플이 크게 보이는 것이 정상이다.

## 3. 📉 평활 커패시터에 따른 출력 특성 분석

### 📐 이론적 배경
* 평활 커패시터는 정류된 출력의 피크 부근에서 충전되고, 피크와 피크 사이 구간에서는 부하를 통해 방전된다.
* 따라서 커패시터 값이 클수록 방전 속도가 느려져 리플 전압이 감소하고, 출력은 더 안정적인 DC 형태에 가까워진다.

### 📊 커패시터 및 주파수 변화에 따른 출력($V_{out}$) 분석

| 주파수 ($f$) | 커패시터 ($C_L$) | 동작 분석 (Region) | 결과 요약 |
| :--- | :--- | :--- | :--- |
| **10 Hz** | $1\\mu F$ | Partial Smoothing | 무커패시터 조건보다 리플 감소, 하지만 여전히 전압 변동이 큼 |
| **1 kHz** | $1\\mu F$ | Smoothing | 출력이 더 안정적으로 유지됨 |
| **10 kHz** | $1\\mu F$ | Improved Smoothing | 리플이 더 작아지고 평균 출력 전압 증가 |
| **10 Hz** | $10\\mu F$ | Strong Smoothing | 리플이 크게 줄고 출력이 DC에 더 가까워짐 |
| **1 kHz** | $10\\mu F$ | Stable DC-like Output | 매우 안정적인 평활 출력 확인 |
| **10 kHz** | $10\\mu F$ | Highly Stable Output | 가장 안정적인 출력 파형 형성 |

> **Note:** 입력 주파수가 높아질수록 커패시터가 더 자주 재충전되므로 리플이 감소하고, 커패시터 값이 커질수록 출력 전압은 더 부드러운 DC 형태에 가까워진다.

### 🔍 트러블슈팅 및 고찰 (Error Analysis) ⭐
1. **커패시터 허용오차:** 실제 커패시터의 정전용량은 표기값과 차이가 있을 수 있어, SPICE보다 리플이 더 크게 보일 수 있다.
2. **주파수 의존성:** 저주파에서는 피크 사이 간격이 길어 커패시터 방전 시간이 길어지므로 리플이 커진다.
3. **비이상적 실험 환경:** 신호 발생기의 내부 저항, 브레드보드의 기생성분, 오실로스코프 및 프로브 측정 오차가 실제 출력 파형에 영향을 줄 수 있다.

## 4. 🧪 SPICE 결과와 실험 결과 비교

### 📌 비교 요약
* 실험 결과와 SPICE 시뮬레이션은 전체적으로 유사한 경향을 보였다.
* 두 결과 모두 전파정류 후 출력이 양의 방향으로만 형성되었고, 커패시터를 추가할수록 리플이 감소했다.
* 또한 입력 주파수가 증가할수록 평균 출력 전압이 증가하고 출력 파형이 더 부드러워지는 경향이 동일하게 관찰되었다.

### 🔍 차이 원인 분석
1. **다이오드 순방향 전압 강하 편차:** 실제 다이오드는 이상소자와 다르게 동작하므로 출력 전압이 더 낮을 수 있다.
2. **기생 저항 및 기생 성분:** 브레드보드 배선과 소자 내부 특성 때문에 실제 파형이 시뮬레이션보다 더 왜곡될 수 있다.
3. **측정 장비 오차:** 프로브 접촉 상태나 오실로스코프 측정 조건에 따라 리플 크기가 다르게 보일 수 있다.

## 5. ✅ 결론 및 느낀 점

* 이번 실험을 통해 전파정류기의 출력 특성은 **입력 주파수**와 **평활 커패시터의 크기**에 크게 의존한다는 것을 확인할 수 있었다.
* 커패시터 값이 $0 \rightarrow 1\\mu F \rightarrow 10\\mu F$로 증가할수록 리플 전압은 감소하고 출력은 더욱 안정적으로 변했다.
* 또한 입력 주파수가 10 Hz에서 1 kHz, 10 kHz로 증가할수록 커패시터가 더 자주 충전되어 출력 전압이 피크값에 더 가깝게 유지되었다.
* 전파정류기는 반파정류기보다 양쪽 반주기를 모두 이용하므로 더 높은 평균 출력 전압과 더 작은 리플을 얻을 수 있다는 점도 이해할 수 있었다.
* 결과적으로, **높은 주파수**와 **큰 커패시터**의 조합이 가장 안정적인 DC 유사 출력 파형을 만든다는 것을 실험적으로 확인하였다.
