## 2. 🧱 Experiment 1: 단일 다이오드 클리퍼 회로 분석

### 📊 측정 조건 (Setup)
* **입력 신호:** $V_{IN} = 5 V_{p-p}$ (100Hz, 1kHz, 100kHz Sine wave)
* **회로 구성:** 단일 다이오드 클리퍼, $R_1 = 100 \Omega$

| 주파수 ($f$) | 동작 상태 (State) | 결과 요약 |
| :--- | :--- | :--- |
| **100 Hz** | Clipping | 양의 피크가 $V_{D,on}$ 수준에서 제한됨 |
| **1 kHz** | Clipping | $V_{IN} > V_{D,on}$ 일 때 출력 전압 제한 확인 |
| **100 kHz** | Clipping | 고주파에서도 양의 반주기 클리핑 동작 유지 |

> **Note:** 다이오드가 ON 상태일 때 출력 전압은 약 $V_{D,on}$으로 제한되며, 나머지 전압은 저항에 인가됩니다.

### 🔍 트러블슈팅 및 고찰 (Error Analysis) ⭐
1. **파형 비대칭성:** 다이오드가 순방향일 때만 도통되므로 양의 피크는 평탄해지지만, 음의 반주기는 거의 그대로 통과함.
2. **저항의 영향:** 실제 실험에서 $100 \Omega$ 저항을 사용하여 다이오드 전류가 증가했으며, 이로 인해 $V_{D,on}$ 값이 이론과 미세한 차이를 보임.

## 3. 📉 Experiment 2: 이중 바이어스 클리퍼 회로 분석

### 📐 회로 설계 및 이론적 배경
* **바이어스 설정:** $V_{B1} = 1V, V_{B2} = 1V$
* **리미팅 범위:** $-(V_{B2} + V_{D,on}) < V_{out} < (V_{B1} + V_{D,on})$

### 📊 주파수에 따른 출력($V_{out}$) 분석

| 주파수 ($f$) | 바이어스 ($V_B$) | 동작 분석 (Region) | 결과 요약 |
| :--- | :--- | :--- | :--- |
| **100 Hz** | $1.0 V$ | Dual Clipping | 상/하단 파형이 대칭적으로 제한됨 |
| **1 kHz** | $1.0 V$ | Symmetrical Limit | 바이어스 전압에 의해 클리핑 레벨 제어 |
| **100 kHz** | $1.0 V$ | Waveform Shaping | 고주파 파형 성형 동작 및 안정성 확인 |

### 🔍 트러블슈팅 및 고찰 (Error Analysis) ⭐
1. **비이상적 특성:** 실제 다이오드의 동작 저항으로 인해 클리핑 지점의 전환(Transition)이 이론보다 부드럽게 나타남.
2. **제어 가능성:** 바이어스 전압 조절을 통해 원하는 전압 레벨에서 파형을 정밀하게 성형할 수 있음을 실험적으로 증명함.
